                    VIDEO FIELDS CLEANUP - QUICK START

📋 WHAT IT DOES
───────────────
Removes all fields from video documents EXCEPT these 15:
  1. approvalStatus      6. durationMinutes    11. storageUrl
  2. approvedAt          7. fileName           12. thumbnailUrl
  3. bunnyVideoId        8. isFree             13. title
  4. contentType         9. order              14. updatedAt
  5. createdAt          10. status             15. views

🚀 HOW TO RUN
─────────────
Method 1 (Easy):
  Double-click: run-video-cleanup.bat

Method 2 (Manual):
  1. Open Command Prompt
  2. cd c:\Users\Administrator\Desktop\E-learning\elearning-app\Bhartam
  3. node cleanup-video-fields.js
